## Setup & Usage

**To run the project locally:**

1. Clone the repository:

    `git clone https://github.com/your-username/memory-mart.git`

    `cd memory-mart`

2. Install dependencies:

    `npm install`

3. Run the development server:

    `npm run dev`

4. Open the application in your browser at [http://localhost:3000](http://localhost:3000).

## Available Scripts

In the project directory, you can run the following commands:

- **`npm run dev`** ‚Äî Starts the development server with hot-reloading.
- **`npm run build`** ‚Äî Compiles the TypeScript files and builds the production bundle.
- **`npm run format`** ‚Äî Formats the code using Prettier.
- **`npm run ci:format`** ‚Äî Checks for code formatting issues without applying changes.
- **`npm run lint`** ‚Äî Runs ESLint to identify potential issues.
- **`npm run lint:fix`** ‚Äî Automatically fixes ESLint issues.
- **`npm run stylelint`** ‚Äî Lints CSS files using Stylelint.
- **`npm run test`** ‚Äî Runs tests using Vitest.

### **Husky and Commit Hooks**

Husky is set up to ensure that code is linted and formatted before committing. The pre-commit hook will automatically run **Lint-Staged**, which runs ESLint and Prettier on your staged files.
